# WarframeVaultStatus

A web tool to check Warframe Void Relics status and market prices.

**Live Demo:** https://buldi.github.io/WarframeVaultChecker/

**Support:** https://buymeacoffee.com/buldidev

## Features

- Search relics and items by name
- View active and vaulted relics
- Real-time market prices from Warframe.Market
- Interactive 3D visualization

## Data Sources

- Relic information from [Warframe Wiki](https://warframe.fandom.com/wiki/Void_Relic)
- Market prices from [Warframe.Market](https://warframe.market/)
